# Teste de Invertibilidade de Matrizes

## Descrição

Este projeto consiste em um programa em Python que realiza o teste de invertibilidade de uma matriz quadrada (2x2 ou 3x3). O programa calcula o determinante da matriz e, caso seja diferente de zero, calcula sua matriz inversa.

## Funcionalidades

* Entrada de matriz 2x2 ou 3x3
* Cálculo do determinante
* Verificação de invertibilidade
* Cálculo da matriz inversa (quando possível)

## Como executar

1. Certifique-se de ter o Python instalado.

2. Instale a biblioteca necessária:
   pip install numpy

3. Execute o programa:
   python matriz_inversa.py

4. Siga as instruções no terminal para inserir os valores da matriz.

## Tecnologias utilizadas

* Python
* NumPy
